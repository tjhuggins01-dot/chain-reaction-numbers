# Chain Reaction Numbers

Deployment-ready browser prototype for a deterministic puzzle game with a headless engine.

## Setup

```bash
npm install
```

## Local Development

```bash
npm run dev
```

## Tests

```bash
npm run test
```

## Typecheck

```bash
npm run typecheck
```

## Lint

```bash
npm run lint
```

## Build

```bash
npm run build
```

## Simulation Harness

```bash
npm run sim
```

## Deploy
GitHub Actions deploys to GitHub Pages on pushes to `main`.
